# 📝 Java Notes Manager (File I/O)

This is a CLI-based Notes App in Java that allows users to write and read notes using FileWriter and FileReader.

---

## 🔧 Features

- Write notes to a `.txt` file
- Read all saved notes
- Text-based menu with clear CLI interface

---

## 💻 How to Run

1. Clone or download the repo
2. Compile the file:
```bash
javac NotesApp.java
